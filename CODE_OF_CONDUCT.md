# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.

The full text is available at:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Reporting

Instances of unacceptable behaviour may be reported confidentially to the project
maintainer at **xent22@gmail.com**. All reports will be reviewed and investigated promptly
and fairly.

The maintainer is obligated to respect the privacy and security of the reporter of any
incident.
